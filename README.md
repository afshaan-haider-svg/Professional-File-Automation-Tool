# Professional File Automation Tool

A modern desktop automation software built with Python and CustomTkinter.

## Features

* Bulk Image Renaming
* File Organizer
* PDF Merger
* File Search Tool
* Duplicate File Finder
* Progress Tracking
* Modern Dashboard UI
* Multi-threading Support

## Technologies Used

* Python
* CustomTkinter
* PyPDF2
* Threading
* File Handling
